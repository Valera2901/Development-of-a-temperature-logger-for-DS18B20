# Development-of-a-temperature-logger-for-DS18B20

Организовать мониторинг температуры в геофизической обсерватории (на примере обсерватории Арти Института геофизики УрО РАН).
Создать устройство на основе микроконтроллера, к которому подключаются несколько температурных датчиков (DS18B20), расположенных в разных местах (вне помещения, внутри помещения, подключен к корпусу измерительного прибора и тд). Устройство должно функционировать автономно, без подключения к компьютеру. При включении питания, устройство должно автоматически записывать принимаемые данные с термодатчиков в файл на SD-карту. Скачивание данных можно организовать несколькими способами: 

1\. при нахождении в сети вайфай контроллер мог бы передавать данные на определенный сайт, 

2\. Можно подключить к микроконтроллеру плату LAN и подключить к локальной сети обсерватории, чтобы данные были доступны как обычный диск в сети.
Проект выполнен на 50% если реализованы следующие функции:
Запись результатов измерений на SD‑карту (без потерь).
Обеспечение актуального точного времени на часах при записи данных в файле
Проект выполнен на 75% если реализованы следующие функции:
Запись результатов измерений на SD‑карту (без потерь).
Обеспечение актуального точного времени на часах при записи данных в файле
просмотр и скачивание файлов данных, без необходимости прерывать измерения (wifi, bt, LAN, веб-интерфейс, сетевой диск, обоснованный выбор командой)
Проект выполнен на 100% если реализованы следующие функции:
Запись результатов измерений на SD‑карту (без потерь).
Обеспечение актуального точного времени на часах при записи данных в файле
просмотр и скачивание файлов данных, без необходимости прерывать измерения (wifi, bt, LAN, веб-интерфейс, сетевой диск, обоснованный выбор командой)
просмотр результатов текущих измерений. Выбрать удобный способ для этого (мобильное приложение, веб-интерфейс)
